# Sistema de Gestion de Subastas
Este es un sistema que permite administrar la venta de cartas. 

## Funcionalidades:

### Gestion de Ventas
1. Ver el listado de ventas por lote, con informacion del contenido; el monto (total y listado); el vendedor; el estado de la venta
[PENDIENTE|CANCELADA|COMPLETADA|DEMORADA].
2. Agregar nuevas ventas.

### Gestion de Compradores
1. Ver el listado de compradores, su informacion de contacto, su estado [ACTIVO|BANEADO]
2. Agregar nuevos compradores.
